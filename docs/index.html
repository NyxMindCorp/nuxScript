nuxScript

Linguagem de programação com funcionalidades exclusivas

Quick Start

# Install
npm i -g nuxscript-lib

# Run
nux run hello.nux

# hello.nux
print("Hello, nuxScript!")

⭐ Features Exclusivas

🔍 Regex Engine

Sintaxe única com match operator, destructuring groups

let pattern = r"/(\w+)@(\w+)\.(\w+)/"
email ~/r "^\w+@\w+\.\w+$"

📊 Query System

LINQ-like queries para dados: select, where, groupBy, join

users |> where(.age > 18)
     |> order_by("name")
     |> select("name")

⚡ Concurrency

Actors, Channels, CSP-style (como Elixir/Erlang)

let counter = actor(fn(state, msg) => ...)
send_to(counter, "inc")

🔧 Meta-programming

Macros, AST manipulation, compile-time evaluation

macro_define("assert", ...)
ast_parse("let x = 1")

📦 Data Structures

Tuple, Set, Dict, Tree, Heap, Graph

let t = tuple(1, 2, 3)
let g = graph()

🏷️ Type System

Refinements, Brands, Newtypes, Opaque types

pos()              // positive number
email()            // valid email

🔣 Custom Operators

Pipeline, null-safe, spaceship, composition

a |> f           // pipe
x ?? default   // coalesce
a <=> b        // spaceship

Installation

# Global (recommended)
npm i -g nuxscript-lib

# Local
npm install nuxscript-lib

Basic Syntax

# Variables
let x = 10           # immutable
var y = 20          # mutable
const MAX = 100

# Functions
fn add(a :: Int, b :: Int) -> Int
    a + b
end

# Structs
struct Point
    x :: Int
    y :: Int
end

# Enums
enum Status
    Pending
    Done(result :: String)
    Failed(error :: String)
end

# Pattern matching
match value
    Ok(v)   -> print("Success: {v}")
    Err(e)  -> print("Error: {e}")
end

# Pipe
[1, 2, 3] |> filter(fn(x) => x > 1)
        |> map(fn(x) => x * 2)
